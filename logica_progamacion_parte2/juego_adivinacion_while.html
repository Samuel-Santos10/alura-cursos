Programa - Juego de adivinacion con ciclo while

Este ejercicio consta de esblecer un numero de intentos que el usuario podra usar en lo que trata de adivinar el numero.
Si el usuario hacerta, le dira que acerto en el numero de intento que acerto.
si falla los 3 intentos, al final le dira que fallo y aparecera el numero correcto que era.

Aprendimos que el ciclo while tiene un comando el cual es "break", en español significa "freno"

El comando "break" se utiliza para finalizar un bucle
